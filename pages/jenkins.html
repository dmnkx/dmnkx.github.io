Jenkins Shared Library Portfolio

두범(doobeom) Jenkins Pipeline
공유 라이브러리

다수 Git 조직·서비스의 빌드·배포를 표준화한 Jenkins Shared Library입니다. Groovy 기반 DSL로 체크아웃부터 CodeDeploy/Lambda 배포, Slack 알림, 승인까지 한 흐름으로 자동화합니다.

회사 두범 (doobeom) 유형 CI/CD 플랫폼 언어 Groovy / Gradle 인프라 AWS · Jenkins

1 프로젝트 개요

두범 Jenkins Pipeline 공유 라이브러리는 Jenkinsfile마다 반복되던 AWS CLI 호출, Git 조작, Slack 웹훅, 배포 대기 로직을 공통 모듈로 추출한 사내 CI/CD 플랫폼입니다.

Jenkins Global Pipeline Library로 등록되며, 각 서비스 Jenkinsfile에서는 @Library('doobeom-lib') _ 한 줄로 로드한 뒤 initialize → buildPipeline(buildConfig(...)) 패턴으로 배포 파이프라인을 구성합니다.

9 사용 예시

최소 패턴은 아래와 같습니다. 실제 서비스는 Closure 안에서 승인·치환·배포를 채웁니다.

@Library('doobeom-lib@master') _

pipeline {
    agent any
    parameters {
        booleanParam(name: 'ACTUAL_DISTRIBUTION', defaultValue: true)
    }
    stages {
        stage('초기 세팅') {
            steps {
                script {
                    initialize([
                        organization: "doobeommarket",
                        repositoryName: "db-market-web",
                        profile: "prod",
                        branch: "master",
                        parameterStoreName: "/doobeom/db-market-web/env/prod",
                    ])
                }
            }
        }
        stage('배포') {
            steps {
                script {
                    buildPipeline(buildConfig(
                        language: "php",
                        gitUrl: "git@github.com:${env.ORGANIZATION}/${env.REPOSITORY_NAME}.git",
                        prepare: { approve.withDeploy() },
                        build:   { sh "composer install" },
                        archive: { sh "zip -r ${env.FILENAME} ." },
                        upload:  { /* s3.cp ... */ },
                        deploy:  { /* deploy.deployAndWait ... */ },
                        completion: { /* git tag */ }
                    ))
                }
            }
        }
    }
    post {
        success { /* slack success */ }
        failure { /* slack failure */ }
        always  { commonPostAlways() }
    }
}
