두범비즈 — B2B Hybrid Android App

판매자/비즈니스를 위한 두범비즈 서비스를 WebView 하이브리드로 제공하는 네이티브 셸. 푸시·어트리뷰션·결제 앱 연동·권한 UX를 담당합니다.

Kotlin Jetpack Compose MVVM WebView Bridge v1.0.9

1. Overview

패키지com.doobeom.biz
타깃minSdk 29 / targetSdk 35
Web Roothttps://m.doobeom.com
유형B2B Hybrid Shell

Kotlin 2.1 · AGP 8.10 · Java 17 · Compose Material3 · Version Catalog
